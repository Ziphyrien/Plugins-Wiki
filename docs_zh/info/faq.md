# ❓常见问题 (FAQ)

## 问：为什么我使用称号后，聊天栏会刷屏消息？

答：经常有人问我这个问题，所以我专门设立了一个页面，对此我很生气。我认为问这个问题的人不尊重我的工作。原因很简单：他们从未浏览过插件的配置文件，也没有阅读过这条消息，更没有浏览过插件的 Wiki。答案是你设置了一个循环动作（circle action），这导致在激活称号后，循环动作内的动作会被重复执行。如果你不想要这个效果，只需在示例前缀配置中删除 `circle-actions` 选项即可。

## 问：为什么称号的条件似乎不起作用？

答：默认情况下，OP 会绕过条件检查。如果你想测试条件检查，请先取消自己的 OP 权限。

## 问：你有称号商店功能吗？

答：没有。90% 的服务器都安装了商店插件，你可以用商店插件做同样的事情，MythicPrefixes 提供这个功能完全是多余的。你可以在前缀条件中设置一个 `permission`（权限）条件，然后使用像 **LuckPerms** 这样的权限管理插件，通过命令给予玩家你在前缀条件中设置的权限。
