# ⚙️安装

## 安装 <a href="#install" id="install"></a>

* 将 `.jar` 文件放入服务器的 `plugins` 文件夹中。
* 停止服务器然后重启。<mark style="color:red;">在服务器启动时无法以任何其他方式加载插件</mark>。
* 更新插件时，请务必删除旧版本。
* 如果你之前使用的是免费版，现在要升级到付费版，只需在服务器上安装付费版并删除免费版即可。插件的配置文件不需要任何更改。
* MythicPrefixes 只是一个 **称号（标签）管理** 插件，<mark style="color:red;">我们不会自动在聊天/Tab列表/计分板中添加前缀</mark>，你必须使用其他插件（如 **TAB**），并在其中使用我们的 [占位符](compatibility.md) 来显示称号。
