# ✅系统要求

## Java 版本

* 基本要求：**Java9+**
* <mark style="color:red;">推荐</mark>使用 **Java 17+**。虽然推荐使用 Java17 及以上版本，但插件是使用 **Java9** 编译的，所以理论上你只需要 Java8 或更高版本。

## 服务器软件

* <mark style="color:red;">推荐</mark>使用 **Paper** 及其下游分支。当插件检测到你的服务器软件是 Paper 时，它将启用一些仅在 Paper 中可用的功能，其中一些功能可以提供细微的性能提升。
* 也支持 **Folia** 服务器。请注意：对 Folia 的支持处于<mark style="color:red;">早期测试阶段</mark>，可能会在正式版本中发布或在未来移除。此支持不作保证。

## 服务器版本

* 插件理论上支持 **1.14** 到 **1.21.7** 之间的任何版本。
* 显然，支持这么多版本并非易事。作者不可能在每次更新插件时都测试 1.14 到 1.21.7 之间的所有版本。如果你在使用特定版本时遇到错误，<mark style="color:red;">请加入我们的 Discord 反馈</mark>。

## 如果你想显示前缀（称号），需要聊天/Tab列表/计分板插件和 PlaceholderAPI

* MythicPrefixes 只是一个 **称号（标签）管理** 插件，<mark style="color:red;">我们不会自动在聊天/Tab列表/计分板中添加前缀</mark>，你必须使用其他插件（如 **TAB**），并在其中使用我们的 [占位符](compatibility.md) 来显示称号。
