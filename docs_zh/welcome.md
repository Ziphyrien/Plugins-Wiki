# 🎉欢迎

## 简介

* MythicPrefixes 是一个轻量级的 Spigot 称号（标签）插件。
* 它与大多数其他称号插件一致，但提供了称号增益（Tag Buff）系统。
* 本插件不会自动将前缀添加到玩家聊天等位置，你需要使用其他插件（如 TAB、Essentials 等），然后使用我们的 PlaceholderAPI 占位符在其中显示前缀！这只是一个前缀管理插件。

## 特性

* 内置称号选择 GUI，支持自定义大小、标题和称号显示物品槽位。
* 内置 ItemFormat 简化版，可为称号显示物品设置自定义模型数据（Custom Model Data）、头颅纹理！
* 支持同时使用多个称号。
* 通过权重系统设置称号的显示排序。
* 支持在称号中使用十六进制颜色和渐变颜色。
* 支持在称号中使用 PlaceholderAPI，在称号中显示小游戏统计数据或其他内容！
* 创建自定义显示占位符，你可以设置黑名单前缀、显示格式和最大显示称号数量等。
* 可以给予玩家额外的 MythicLib (MMOLib) 属性。
* 可以给予玩家 libreforge 效果，做其他 eco 风格插件能做的事情。例如对末影龙造成 50% 更多伤害！
* 内置条件系统，可以根据条件决定是否解锁称号以及同时使用的最大称号数量。
* 当玩家装备或卸下称号时运行动作或循环动作。（你可以设置玩家装备称号后自动装备装饰品）
* 还有更多！
