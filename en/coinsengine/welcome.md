**CoinsEngine** is a lightweight, highly customizable economy & custom virtual currency provider with built-in [Vault integration](hooks/vault.md).

No need in different economy, coins, points and tokens plugins anymore, CoinsEngine will do everything for you!

## Features

* [**Unlimited Currencies**](features/currencies.md). Create as many currencies as you want.
* [**Data Migration**](features/migration.md). Migrate player balances from other plugins with a single command.
* [**Cross-Server**](features/cross-server.md). Share currency balances between your servers.
* [**Plugin Integrations**](hooks/plugins.md). CoinsEngine is supported by many popular plugins.
* [**Vault**](hooks/vault.md) Support. Set desired currency to handle your economy.
* [**PlaceholderAPI**](placeholders/papi.md) Support.
* **SQL Database**. With SQLite and MySQL support.
* **Decimals & Integers**. Create decimal and integer currencies.
* **Start Value**. Set start currency value for new players.
* **Max. Value**. Set max. currency value for player balance.
* **Payments**. Send currency to other players. Toggleable per currency.
* [**Amount Shortcuts**](https://nightexpressdev.com/nightcore/configuration/number-formation/#number-shortcuts). Provide large amounts in commands by using aliases.
* [**Wallet**](features/wallet.md). View balance of all currencies with a single command!
* [**Currency Prefix**](features/prefix.md). Set custom currency prefix for all currency messages.
* [**Currency Format**](features/prefix.md). Set custom format for each currency.
* [**Currency Exchange**](features/exchange.md). Exchange one currency for another.
* [**Currency Commands**](commands.md#currency-commands). Create direct, custom commands for your currencies.
* [**Economy Commands**](commands.md#dedicated-commands). Provides customizable `/pay`, `/balance` and `/balancetop` commands.
* [**Permission Requirement**](permissions.md). Restrict currency access by permissions.
* [**Leaderboards**](features/leaderboards.md). View top players by their currency balance.
* [**Opertation Logs**](features/logs.md). Log all currency operations into separate file.

## System Requirements

* Server Software: [**Spigot**](https://www.spigotmc.org/link-forums/88/) or [**Paper**](https://papermc.io/downloads/paper)
* Server Version: **1.21.4** or above
* Java Version: [**21**](https://adoptium.net/temurin/releases) or above
* Dependencies:
  + [**nightcore**](https://nightexpressdev.com/nightcore/) - Plugin engine.
* Optional Plugins:
  + [**Vault**](https://spigotmc.org/resources/34315/) - Economy integration.
* Folia Supported: **No**
* Forge Supported: **No**
